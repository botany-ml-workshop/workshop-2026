# Machine Learning for Plant Trait Extraction from Images

## Workshop Overview

This workshop introduces the principles of computer vision and machine learning for extracting biological traits from images. Participants will learn the basics of image annotation, semantic segmentation, model training, and model evaluation using plant photographs from iNaturalist and herbarium collections.

By the end of the workshop, participants will have trained and evaluated computer vision models and explored how these approaches can be used to generate trait datasets at scale.
